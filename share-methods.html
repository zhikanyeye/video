🚀 选择分享方式

为您的视频播放列表选择最适合的云端分享方案

推荐
🏠
自建数据库
使用本地Node.js服务器和SQLite数据库存储播放列表，简单快捷，无需任何授权。

✅ 优点

  • 无需任何账号和密钥
  • 完全免费使用
  • 数据完全自主控制
  • 响应速度快
  • 支持统计和管理

⚠️ 注意

  • 需要运行本地服务器
  • 数据存储在本地
  • 需要端口转发才能外网访问
🐙
GitHub Gists
使用GitHub Gists作为云端数据库，全球CDN加速，永久免费存储。

✅ 优点

  • 完全免费的云端存储
  • 全球CDN加速访问
  • 自动版本控制
  • 支持多设备同步
  • 数据永久保存

⚠️ 注意

  • 需要GitHub账号
  • 需要创建访问令牌
  • 分享的播放列表公开可见
  • 依赖GitHub服务稳定性
功能对比 自建数据库 GitHub Gists
设置难度 简单 (一键启动) 需要授权 (需要Token)
数据安全 本地存储 公开可见
访问速度 本地最快 全球CDN
多设备同步 仅本地 完美同步
外网分享 需要配置 直接分享
存储限制 无限制 足够使用
